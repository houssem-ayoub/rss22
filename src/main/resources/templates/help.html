Accueil Liste des articles
Liste des requêtes:
URI Methode Résumé de l’Opération
/ GET Affiche la page d’accueil sous format HTML
/help GET Affiche la page contenant les informations d’aide
/rss22/resume/xml GET Affiche la liste des articles stockés sous format XML
Retour:
<items>
    <item>
        <guid>{guid}</guid>
        <title>{title}</title>
        <date>{date}</date>
    </item>
...
</items>
/rss22/resume/html GET Affiche la liste des articles stockés sous format HTML
/rss22/xml/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid} sous Format XML
/rss22/html/{guid} GET Affiche le contenu complet de l’article dont l’identifiant est {guid} sous Format HTML
rss22/insert POST stocker Les articles du flux dans la base de données.
Transmis: un flux XML qui respècte le schéma rss22.xsd
Retour:
Un flux xml sous la forme :
<respone>
    <guid>{guid}</guid>
    <status>INSERTED</status>
    <description>{message}</description>
</respone>
En cas d’échec de l’opération,
<respone>
    <status>ERROR</status>
    <description>{message}</description>
</respone>
/rss22/delete/{guid} DELETE Suppression de l’article dont l’identifiant est {guid}.
tout flux rss22 vide sera supprimé. Retour:
Un flux xml sous la forme :
<respone>
    <guid>{guid}</guid>
    <status>DELETED</status>
    <description>{message}</description>
</respone>
En cas d’échec de l’opération,
<respone>
    <status>ERROR</status>
    <description>{message}</description>
</respone>
/rss22/search?... GET Rechercher la liste des articles répondant aux contraintes exprimées dans la requête ◦ date [YYYY] Liste des articles dont la date est identique, ou postérieure à celle indiquée dans la requěte. ◦ titreListe [mot1;...] des articles dont le titre comprend l'un des mots de la requête.
Retour:
<items>
    <item>
        <guid>{guid}</guid>
        <title>{title}</title>
        <date>{date}</date>
    </item>
...
</items>
